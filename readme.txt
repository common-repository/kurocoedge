== Description ==

= What is KurocoEdge =

A 'behind-the-scenes' helper that provides a Programmable Proxy, as well as a variety of common edge computing solutions.

**Edge CDN**

KurocoEdge's CDN is built on powerful Fastly's infrastructure. It is a highly scalable and reliable solution. This provides with Caching ability for high speed delivery for your content. KurocoEdge provides a smart and optimized solution to caching your data. You can invalidate the cache at the click of a button or configure this plugin to automatically do that whenever there is a content change. So, you can focus on what is important, your content!





== Getting Started ==

This section includes steps required get set up with the KurocoEdge plugin for WordPress.

= Prerequisite =

* A Kuroco environment. You can get that [here](https://kuroco.app/free_trial).
* Your API URL, Site Key and Edge Key.

= Setup =

Once you have the Prerequisites ready, you can install the plugin and configure the information there. Once saved, the plugin will automatically connect to the Kuroco environment.
